Extensão Teste

Uma extensão simples para o Google Chrome (Manifest V3) criada para o Bootcamp II.

O que ela faz?

Esta extensão é um projeto de aprendizado que demonstra os fundamentos do desenvolvimento de extensões para o Chrome com o Manifest V3. Ela inclui:

  • Um popup interativo com botões.
  • Um service worker que roda em segundo plano.
  • Um content script que modifica páginas da web.
  • Comunicação entre o popup e o service worker.
